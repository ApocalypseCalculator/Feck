File "indices" are stored inside data.db by the following schema: 

name TEXT NOT NULL,
id TEXT NOT NULL UNIQUE,
date INTEGER NOT NULL,
type TEXT NOT NULL DEFAULT "standard"

name is the full file name with extension
id is a unique id generated by UUIDv4
date is an integer representing the upload time in ms
type specifies general file type (can be expanded)

Uploads are formatted into uploads/fileid/filename.extension
